## Project Overview

Gym Management System is a project that will be implemented as a software system
accessible through a web interface using a browser. It will act as a self-contained web
site that manages data used in operations of a gym, staffed by various employees and
used by customers who are registered as members. Gym Management System will
provide user interfaces for both employees and members. For employees it will act as a
replacement of a set of spreadsheets and documents, paper or electronic, that would be
used to maintain gym-related data in the absence of such system, such as members and
employees lists, information about classes and schedules.

## Features

### Members
- Register and log in with role-based access
- Browse and search fitness classes by type, trainer, or date
- Book available class sessions
- Join waitlist when a class is full
- View personal attendance history
- Manage membership plan and payments
- Leave reviews for trainers

### Trainers
- Create and manage fitness classes
- Schedule and update class sessions
- View enrolled members and attendance

### Administrators
- Manage membership plans and pricing
- Track and monitor payments
- Manage equipment inventory
- Add, update, or remove users
- Full access to all system data
